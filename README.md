# Rede Sarelli - Gestão de Campanha Política

Sistema avançado para gestão de lideranças, fiscais e eleitores, com integração financeira e analytics.

## 🚀 Tecnologias
- **Frontend**: React + Vite + TypeScript
- **UI**: Tailwind CSS + Radix UI (Shadcn/ui)
- **Backend/DB**: Supabase
- **PWA**: Suporte para instalação e modo offline
- **Analytics**: Vercel Analytics & Speed Insights

## 📋 Funcionalidades Principais
- **Hierarquia de Usuários**: Gestão de Master, Coordenadores, Suplentes e Lideranças.
- **Cadastros Políticos**: Fluxo de cadastramento de lideranças e fiscais com geolocalização.
- **Financeiro**: Controle de contas a pagar e fornecedores.
- **Integração Social**: Captura de menções e interações no Instagram.
- **Site Público**: Galeria de fotos e formulários de contato integrados.

## 🛠️ Desenvolvimento
1. Clone o repositório
2. Instale as dependências: `npm install`
3. Configure as variáveis de ambiente no `.env`
4. Inicie o servidor: `npm run dev`
